### CONTRIBUTING

#### Initial Setup

* clone this repo
* `cd` into it
* exec: `bundle config build.nokogiri --use-system-libraries`
* exec: `bundle install` or `sudo bundle install`

#### Previewing Edits in Jekyll

* `cd` into the repo
* exec: `rake run`
* follow the instructions to access the site locally
